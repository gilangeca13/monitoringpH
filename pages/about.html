Deskripsi Sistem

Proyek Internet of Things (IoT) ini dirancang khusus untuk memantau tingkat keasaman (pH) air Danau Maninjau secara real-time. Dengan integrasi sensor pH dan ESP32, data dikirim secara nirkabel melalui modul SIM800L ke cloud platform ThingSpeak, membantu para petani ikan dan pihak terkait memantau kelayakan air dan mencegah kegagalan budidaya ikan.

Tujuan Proyek

  • Memonitor kualitas air Danau Maninjau secara real-time dan akurat.
  • Mendeteksi kondisi air secara instan apakah berada pada kondisi asam, normal, atau basa.
  • Mengirimkan pembacaan data secara terjadwal ke server cloud ThingSpeak.
  • Memberikan peringatan dini dan notifikasi status air untuk tindakan pencegahan cepat.

Komponen Sistem

  • ESP32: Mikrokontroler utama dengan modul Wi-Fi dan Bluetooth built-in.
  • Sensor pH Air: Mengukur konsentrasi ion hidrogen dalam air.
  • SIM800L: Modul seluler GSM/GPRS untuk pengiriman data tanpa ketergantungan Wi-Fi lokal.
  • OLED SSD1306: Layar mini untuk pemantauan data langsung di lokasi alat dipasang.
  • ThingSpeak: Platform IoT cloud terpercaya untuk analisis data dan visualisasi grafik.

Cara Kerja Sistem

  1. Sensor pH membaca tingkat keasaman sampel air danau.
  2. ESP32 memproses sinyal analog sensor menjadi nilai numerik pH yang valid.
  3. Data pH ditampilkan pada layar OLED fisik di lokasi pemantauan.
  4. ESP32 menggunakan modul SIM800L untuk mengirimkan data ke ThingSpeak via jaringan GSM.
  5. Website ini menarik data feed dari ThingSpeak secara otomatis untuk divisualisasikan kepada pengguna.

Status Sistem

Sistem monitoring ini beroperasi secara penuh dan terus-menerus mengirimkan data pembaruan baru setiap 15 detik ke cloud server.
