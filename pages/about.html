Deskripsi Sistem

Proyek Internet of Things (IoT) ini dirancang khusus untuk memantau tingkat keasaman (pH) air Danau Maninjau secara real-time. Dengan integrasi sensor pH dan ESP32, data dikirim secara nirkabel melalui modul SIM800L ke cloud platform ThingSpeak.

Tujuan Proyek

  • Memonitor kualitas air Danau Maninjau secara real-time dan akurat.
  • Mendeteksi kondisi air secara instan apakah berada pada kondisi asam, normal, atau basa.
  • Mengirimkan pembacaan data secara terjadwal ke server cloud ThingSpeak.
  • Memberikan peringatan dini dan notifikasi status air untuk tindakan pencegahan cepat.

Status Sistem

Sistem monitoring ini beroperasi secara penuh dan terus-menerus mengirimkan data pembaruan baru setiap 15 detik ke cloud server.
